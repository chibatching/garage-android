Garage for Android
=================

[![Release](https://jitpack.io/v/tokubai/garage-android.svg)](https://jitpack.io/#tokubai/garage-android)

# setup

`build.gradle`

```groovy
allprojects {
    repositories {
        jcenter()
        maven { url "https://jitpack.io" }
    }
}
```

`module/build.gradle`

```groovy
dependencies {
    implementation "com.github.tokubai:garage-android:$version"
}
```
